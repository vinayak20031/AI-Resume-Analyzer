# AI Resume Analyzer

AI Resume Analyzer is a web application that allows users to upload their resumes and get feedback on skills, keywords, and overall resume quality.

The system analyzes the resume and helps users improve their chances of getting shortlisted for jobs.

---

##  Live Demo
https://ai-resume-analyzer-production-9176.up.railway.app

---

##  Features

- User Registration and Login
- Secure authentication using JWT
- Upload Resume
- Resume analysis and feedback
- Resume history tracking
- Clean and responsive UI

##  Installation (Run Locally)

Clone the repository

```
git clone https://github.com/vinayak20031/AI-Resume-Analyzer.git
```

Go into the project folder

```
cd AI-Resume-Analyzer
```

Install dependencies

```
npm install
```

Create `.env` file

```
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

Run the server

```
npm start
```

Open in browser

```
http://localhost:8080
```
